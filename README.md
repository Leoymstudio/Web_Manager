# 个人收藏夹整理应用

一个功能完整的现代化书签管理工具，基于Vue.js构建，支持树形分类、卡片展示、搜索过滤、拖拽排序等功能。

## 🌟 主要特性

### 📁 智能分类管理
- **树形分类结构**：支持多级文件夹分类，可展开/收起
- **分类统计**：实时显示每个分类下的书签数量
- **快速切换**：点击分类快速过滤书签

### 🔍 强大搜索功能
- **实时搜索**：输入关键词即时过滤书签
- **多字段匹配**：支持标题、描述、URL、标签搜索
- **标签过滤**：点击标签云快速过滤相关书签

### 🎨 现代化界面
- **响应式设计**：完美适配桌面、平板、手机
- **卡片式展示**：直观的网格布局，支持网格/列表切换
- **优雅动画**：流畅的过渡效果和悬停动画

### ✨ 书签管理
- **CRUD操作**：完整的增删改查功能
- **拖拽排序**：支持书签拖拽重新排序
- **访问统计**：记录书签访问次数和最后访问时间

### 🏷️ 标签系统
- **灵活标签**：为书签添加多个标签
- **标签云**：可视化显示热门标签
- **颜色编码**：不同标签使用不同颜色区分

### 📤 数据管理
- **导入导出**：支持Chrome书签HTML格式和JSON格式
- **本地存储**：数据自动保存到浏览器本地存储
- **数据备份**：一键备份和恢复功能

## 🚀 快速开始

### 本地运行
1. 下载所有文件到本地文件夹
2. 在项目根目录运行本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问：`http://localhost:8000`

### 使用方法
1. **添加书签**：点击"添加书签"按钮，填写信息保存
2. **管理分类**：在左侧边栏创建和管理分类
3. **搜索书签**：使用顶部搜索框快速查找
4. **导入书签**：支持导入Chrome导出的HTML文件

## 🛠️ 技术栈

- **Vue.js 3**: 渐进式JavaScript框架
- **Tailwind CSS**: 实用优先的CSS框架
- **Anime.js**: 动画库
- **Pixi.js**: 2D渲染引擎（背景粒子效果）
- **ECharts**: 数据可视化
- **LocalStorage**: 本地数据存储

## 📱 响应式设计

- **桌面端** (≥1024px): 三栏布局，完整功能展示
- **平板端** (768px-1023px): 可折叠侧边栏，双列网格
- **移动端** (<768px): 抽屉式侧边栏，单列布局

## ⌨️ 快捷键

- `Ctrl/Cmd + K`: 聚焦搜索框
- `Ctrl/Cmd + N`: 添加新书签
