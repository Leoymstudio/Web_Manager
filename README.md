# 个人收藏夹整理应用
version 0.5
一个功能完整的现代化书签管理工具，基于Vue.js构建，支持树形分类、卡片展示、搜索过滤、拖拽排序等功能，提供本地存储和SQLite数据库双重支持。

## 🌟 主要特性

### 📁 智能分类管理
- **树形分类结构**：支持多级文件夹分类，可展开/收起
- **分类统计**：实时显示每个分类下的书签数量
- **快速切换**：点击分类快速过滤书签
- **拖拽排序**：支持分类拖拽重新排序

### 🔍 强大搜索功能
- **实时搜索**：输入关键词即时过滤书签
- **多字段匹配**：支持标题、描述、URL、标签搜索
- **标签过滤**：点击标签云快速过滤相关书签

### 🎨 现代化界面
- **响应式设计**：完美适配桌面、平板、手机
- **卡片式展示**：直观的网格布局，支持网格/列表切换
- **优雅动画**：流畅的过渡效果和悬停动画

### ✨ 书签管理
- **CRUD操作**：完整的增删改查功能
- **拖拽排序**：支持书签拖拽重新排序和跨分类移动
- **访问统计**：记录书签访问次数和最后访问时间
- **批量操作**：支持多选书签进行批量移动、删除操作

### 🏷️ 标签系统
- **灵活标签**：为书签添加多个标签
- **标签云**：可视化显示热门标签
- **颜色编码**：不同标签使用不同颜色区分

### 💾 数据存储
- **双重存储**：支持浏览器localStorage和SQLite数据库
- **自动同步**：前端与后端数据自动同步
- **数据迁移**：支持从localStorage一键迁移到数据库

### 📤 数据管理
- **导入导出**：支持Chrome书签HTML格式和JSON格式
- **智能去重**：导入时自动检测并合并重复书签
- **分类映射**：导入时智能匹配和创建分类结构
- **数据备份**：一键备份和恢复功能

## 🚀 快速开始

### 后端启动
1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动后端服务器：
   ```bash
   node .\server.js
   ```
   服务器将在 http://localhost:3000 启动

### 前端访问
1. 在浏览器中访问：`http://localhost:3000`
2. 或者使用静态服务器（仅localStorage模式）：
   ```bash
   npx http-server . -p 8080
   ```
   然后访问：`http://127.0.0.1:8080`

## 🛠️ 技术栈

### 前端技术
- **Vue.js 3**: 渐进式JavaScript框架
- **Tailwind CSS**: 实用优先的CSS框架
- **Anime.js**: 动画库
- **Pixi.js**: 2D渲染引擎（背景粒子效果）
- **ECharts**: 数据可视化

### 后端技术
- **Express.js**: Node.js Web应用框架
- **Better-SQLite3**: 高性能SQLite数据库驱动
- **CORS**: 跨域资源共享支持
- **Body-parser**: 请求体解析中间件

### 数据存储
- **LocalStorage**: 浏览器本地存储（前端模式）
- **SQLite**: 轻量级数据库（后端模式）

## 📱 响应式设计

- **桌面端** (≥1024px): 三栏布局，完整功能展示
- **平板端** (768px-1023px): 可折叠侧边栏，双列网格
- **移动端** (<768px): 抽屉式侧边栏，单列布局

## 🔧 API接口

### 书签管理
- `GET /api/bookmarks` - 获取所有书签
- `POST /api/bookmarks` - 创建新书签
- `PUT /api/bookmarks/:id` - 更新书签
- `DELETE /api/bookmarks/:id` - 删除书签

### 分类管理
- `GET /api/categories` - 获取所有分类
- `POST /api/categories` - 创建新分类
- `PUT /api/categories/:id` - 更新分类
- `DELETE /api/categories/:id` - 删除分类

### 数据管理
- `GET /api/snapshot` - 获取完整数据快照
- `POST /api/snapshot` - 保存数据快照
- `POST /api/import` - 导入数据（支持合并）
